## 🩺 Featured Project — WellCheck

<div align="center">
<img src="https://media.giphy.com/media/L1R1tvI9svkIWwpVYr/giphy.gif" width="220">
</div>

**WellCheck** is a cross-platform smart health monitoring system I built for my final year project — published in the **International Journal of Innovative Computing (IJIC)**. It tackles a real gap in Malaysian healthcare: no existing platform combines live sensor data, AI diagnosis support, and a consistent experience across web and mobile.

🔧 **How it's built:**

![Spring Boot](https://img.shields.io/badge/Spring%20Boot-6DB33F?style=for-the-badge&logo=springboot&logoColor=white)
![Flutter](https://img.shields.io/badge/Flutter-02569B?style=for-the-badge&logo=flutter&logoColor=white)
![MongoDB](https://img.shields.io/badge/MongoDB%20Atlas-47A248?style=for-the-badge&logo=mongodb&logoColor=white)
![Django](https://img.shields.io/badge/Django-092E20?style=for-the-badge&logo=django&logoColor=white)
![Arduino](https://img.shields.io/badge/Arduino-00979D?style=for-the-badge&logo=arduino&logoColor=white)
![Stripe](https://img.shields.io/badge/Stripe-635BFF?style=for-the-badge&logo=stripe&logoColor=white)

- 🌐 **Web app** (Spring Boot + Thymeleaf, MVC) + 📱 **Mobile app** (Flutter, MVVM) — synced live through a shared MongoDB Atlas database
- 📡 **Arduino sensors** stream real-time heart rate, temperature, SpO2, and ECG readings straight into the system
- 🤖 **AI-powered disease prediction API** (Django + Python) gives doctors extra diagnostic support
- 👥 Five user roles — Patient, Doctor, Pharmacist, Clinic Assistant, Admin — each with their own dashboard
- 💳 Built-in appointments, prescriptions, billing (Stripe), and email notifications
- ✅ Validated with **16 black-box test cases** across 5 modules + **42 User Acceptance Test cases** with a real clinic stakeholder — 100% pass

> WellCheck was built to close the gap that apps like MySejahtera and MyHealth360 leave open: no real-time sensor monitoring, no AI-assisted prediction, and inconsistent web/mobile experiences.

## 🤖 Machine Learning & Research

<div align="center">

I'm especially drawn to **AI applied to healthcare** — using models to catch problems earlier and support (not replace) clinical judgment.

- 🧠 **AlzPredict: Neural Network Approach to Early Diagnosis** — *IJIC* — an MLP (multi-layer perceptron) classifier for early Alzheimer's risk detection
- 🩺 **WellCheck's AI Prediction API** — a Django/Python disease-prediction service that analyzes patient vitals to flag risk and support doctor decisions

Both push toward the same idea: healthcare tech that's proactive instead of reactive. 💊
